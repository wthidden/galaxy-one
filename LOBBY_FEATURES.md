# StarWeb Lobby Features

## Game Lists

### My Games (Left Panel)
Games you've already joined as a player:
- Shows your active games
- Displays your character type and name
- **"Enter Game"** button to start playing
- Click on a game to see details and scoreboard

### Available Games (Left Panel)
Games created by other players that you can join:
- Shows games you're NOT in yet
- Only displays games that aren't full
- **"Join"** button to join the game
- Click on a game to see details before joining

## Game Details (Middle Panel)
Click any game in either list to view:
- Game status (waiting/active/completed)
- Current turn number
- Player count (e.g., "3/6 players")
- When the game was created
- **Scoreboard** - See all players, their characters, and scores

## Lobby Chat (Right Panel)
Real-time chat with all players in the lobby:
- See who's online and looking for games
- Coordinate game starts
- Chat persists across page refreshes
- Messages are color-coded (your messages highlighted)
- Auto-scrolls to latest messages

## Admin Messages
Important announcements from server administrators appear in a banner at the top of the lobby (when active).

## Creating a Game
1. Click **"+ Create Game"**
2. Enter game details:
   - **Game Name** - What others will see
   - **Your Character Type** - Choose your strategy
   - **Your Character Name** - Your in-game identity
   - **Max Players** - 2-12 players
   - **Map Size** - Classic (255 worlds) recommended
3. Click **"Create Game"**
4. You'll automatically enter the game
5. Other players can see and join from "Available Games"

## Joining a Game
1. Browse **"Available Games"**
2. Click a game to see details
3. Click **"Join"**
4. Choose your character type and name
5. Click confirm
6. You'll automatically enter the game

## Map Size Options
- **Small (50 worlds)** - Quick games, tight quarters
- **Medium (100 worlds)** - Balanced gameplay
- **Large (150 worlds)** - More room to expand
- **Huge (200 worlds)** - Long-term strategy
- **Classic (255 worlds)** - Original StarWeb experience ‚≠ê Default

## Scoreboard Details
When viewing game details, the scoreboard shows:
- **Rank** - Current standing
- **Player** - Account name
- **Character** - Character type and name
- **Score** - Current points

The scoreboard updates as the game progresses.
